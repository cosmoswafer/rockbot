# Top 10 User Stories — RockBot

> **Priority**: P0 = critical, P1 = core, P2 = enhancement

1. **DM Conversation** (P0) — Team member DMs bot, gets AI replies with conversation history across sessions.
2. **@Mention in Channels** (P0) — Channel member `@rockbot`, gets reply in-thread with per-room isolation.
3. **Web Search via Exa** (P1) — User asks bot to search web; returns summarized results with URLs.
4. **URL Fetch and Read** (P1) — User shares URL; bot fetches and returns as raw, markdown, or JSON.
5. **Image Generation (fal.ai)** (P2) — User prompts image; bot generates, saves to WebDAV, returns URL.
6. **Calendar Events (CalDAV)** (P2) — User creates/lists/updates/deletes calendar events via chat.
7. **Knowledge Persistence** (P2) — User tells bot to remember/recall/forget facts; persisted to WebDAV.
8. **Permanent Soul Memory** (P1) — User sets preferences; bot adapts permanently across restarts.
9. **WebDAV File Operations** (P1) — User reads/writes/lists/edits files in bot's WebDAV workspace.
10. **Vision / Image Analysis** (P2) — User shares image URL; bot returns metadata (MIME, dimensions).
